Interactive Elements

In-Text link

CSS selector:

.in-text-link

Sample code:

<a class="in-text-link" href="#">Default Link</a>

Rendered element:

Default Link

Buttons

CSS selector

".cta-button" or ".cta-prj-button"

Sample code:

              <button class="cta-button page-scroll" type="button" data-toggle="collapse" data-target="#toggleStudious" aria-expanded="false" aria-controls="toggleStudious">MORE DETAILS</button>

Rendered element:

Text Styling

Headings

CSS selectors:

h1, h2, and h3 element tags

Sample code:

                
                  <h1>header 1</h1>
                  <h2>header 2</h2>
                  <h3>header 3</h3>

Rendered element:

Heading1

Heading2

Heading3

Bulleted List

CSS selector:

ul element tag

Sample code:

                
                  <ul class="styling-bullet">
                  <li>Apple</li>
                  <li>Orange</li>
                  <li>Grape</li>
                  </ul>

Rendered element:

  • Apple
  • Orange
  • Grape

Numbered List

CSS selector:

ol element tag

Sample code:

                
                  <ol class="styling-ordered">
                  <li>Item A</li>
                  <li>Item B</li>
                  <li>Item C</li>
                  </ol>

Rendered element:

  1. Item A
  2. Item B
  3. Item C

<nav class="navbar navbar-light navbar-expand-sm" id="navbar-bg">
                <div class="container">
                  <a href="index.html" class="nav-item nav-link active navbar-brand" id="navLogo">

                  </a>
                  <button class="navbar-toggler" type="button" data-toggle="collapse" data-target="#toggleNavBar" aria-controls="toggleNavBar" aria-expanded="false" aria-label="Toggle navigation">

                  </button>
                  <div class="collapse navbar-collapse" id="toggleNavBar">
                    <div class="navbar-nav ml-auto">
                      <a href="projects.html" class="nav-item nav-link">WORK</a>

                  </div>
                </div>
              </nav>

WORK ABOUT
